# Sort module

Aisle-order (Gangfolge) calculation. Not implemented in this directory.

Sorting logic lives in:
- `src/lib/store/aisle-order.ts` – per-store aisle order from pairwise data
- `src/lib/store/hierarchical-order.ts` – hierarchical ordering
- `src/lib/list/list-helpers.ts` – `sortAndGroupItems()` for list display
